# Elite Talent & Capacity Command

A React workforce-planning workspace for **Elite Era Development L.L.C.** It links team skills, weekly capacity, project staffing, hiring choices, contractor options, and delivery risk.

## Workflow

```text
Project demand
→ Team utilization and skill coverage
→ Capacity gap and staffing risk
→ Internal rebalance or contractor request
→ Hiring decision and approval
→ Recruiting pipeline
```

## Features

- Capacity dashboard for delivery leaders
- Project staffing planner
- Team workload and skill matrix
- Overload and burnout signals
- Skill-gap detection
- Hire versus contractor cost comparison
- Finance and Founder role approvals
- Candidate pipeline tracker
- TXT, JSON, and print/PDF exports
- Browser local storage, engine tests, and GitHub Actions

## Run locally

```bash
npm install
npm run dev
```

Open the address shown in the terminal, normally `http://localhost:5173`.
