# Workforce analytics ethics and governance policy

## Purpose

This project supports planning research: skills, capacity, demand, learning paths, hiring gaps, burnout-risk proxies, and contractor alternatives.

## Prohibited automated decisions

The system must not automatically make or recommend final decisions about termination, promotion, compensation, discipline, health status, or protected-class treatment.

## Required review before real use

Real use requires data authorization, employee privacy review, HR/legal review, bias testing, documentation of model limits, opt-out or appeal procedures where applicable, and human accountability.

## Burnout-risk boundary

Burnout risk in this repository is a workload/engagement planning proxy. It is not a medical, psychological, or performance diagnosis.

## Fairness boundary

Synthetic subgroup gaps are regression-test signals. Real fairness analysis must use legally and ethically appropriate data, statistical controls, and expert review.
